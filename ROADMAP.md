# Roadmap

kakunin-mcp is early and developed in the open. This roadmap is directional, not a
commitment — issues and PRs are welcome on any of it.

## Now
- Stabilize core MCP server functionality and tool coverage.
- Harden error handling and edge-case testing.

## Next
- Additional tools for common workflows.
- SSE transport support.
- Richer error surfaces for easier debugging by integrators.

## Exploring
- Deeper integration with kakunin-core as its identity primitives mature.

Have an idea or want to help shape direction? Open an issue — that's how we propose
and discuss what's next.
